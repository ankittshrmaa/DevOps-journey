## Tech for Website: TaskTeddy:---->
### Frontend Stack

```
React 18 + TypeScript + Vite
```

- **React 18** - UI library with concurrent features
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server

### Styling

```
Tailwind CSS 3.4 + shadcn/ui
```

- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Component library built on Radix UI primitives
- **Custom CSS Variables** - For theming (colors, fonts, gradients)

### Routing & State

```
React Router DOM 6 + TanStack Query
```

- **React Router** - Client-side routing
- **TanStack Query** - Server state management (configured but not actively used)

### Deployment

```
Docker + Nginx
```

- **Multi-stage Docker build** - Node.js builder + Nginx production image
- **Nginx** - Web server with gzip, caching, and security headers
- **SPA routing** - All routes serve index.html

### CI/CD

```
Jenkins
```

- ** Use jenkinsfile to deploy my code direcly to AWS by Automation
- ** Jenkins help with 0-Downtime for website deployment.

### Hosting 

```
AWS + CloudFlare
```

- **I use AWS for hosting my Website.
- **CloudFlare for DNS.
