# GitHub webhook that will tell Jenkins when I push code to GitHub.

Using this Web-hook Jenkins know everytime that there is code push to the repo. https://github.com/ankittshrmaa/anantglobal


# Config Webhook
1. GitHub repo. Anantglobal
2. Settings--> Webhook
3. New webhook
4. Payload URL: enter http://publicip-jenkins/github-webhook/
5. Content type: application/json
6. secret none.
7 Just the push event
