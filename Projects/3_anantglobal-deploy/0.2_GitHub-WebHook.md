# GitHub webhook that will tell Jenkins when I push code to GitHub.

Using this Web-hook Jenkins know everytime that there is code push to the repo. https://github.com/ankittshrmaa/anantglobal


# Config Webhook
1. GitHub repo. Anantglobal
2. Settings--> Webhook
3. New webhook
4. Payload URL: enter http://publicip-jenkins/github-webhook/
5. Content type: application/json
6. secret none.
7 Just the push event
save

#### If there is green tick, means github able to ping jenkins.

### Test:
In VS-code pull the actual code and add jenkinsfile, this file tell jenkins that what to do.
pipeline {
    agent any
    stages {
        stage('Test Webhook') {
            steps {
                echo 'Webhook is working! Code received from GitHub.'
                sh 'ls -la'
            }
        }
    }
}

#### Push the code to git again and then github webhook trigger the jenkins and Test job runs
