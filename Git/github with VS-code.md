# For use Github on Vscode.
Download git extension on VS-Code. Connect Vs-code with github.

Confirm that you are connected with github by vscode in setting>user

## Pull:-

To push and pull use terminal
1. Clone Repo. 
git clone https://github.com/ankittshrmaa/DevOps-journey.git

2. Repo is show on the file you had open
3. check git status by terminal
git status

4. edit the code, write the code, if you have already write the code copy that code to repo. location then 

## Push:-
1. git .   
2. git init
3. git commit -m "message"
4. git push -u origin main --force


## Command use for:-
1. git .: for select the files we edit . refer to all files in th repo. 
2. git init : for add or confirm the edits
3. git commit -m for logs and track every push or pull by git
4. git push -u origin main : For Push the final code to github repo.
5. git status: check the git status if somthing change in the code or not
5. git remote -v : Check if VS- code connected to the repo. by Push and Pull
6. git clone name : To cpoy the whole repo from guthub to VSCode or location
