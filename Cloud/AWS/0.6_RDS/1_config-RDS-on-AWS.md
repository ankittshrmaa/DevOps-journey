# Setup RDS-PostgreSQL on AWS.
## In this i will make a postgreSQL RDS on a VPC with private subnet and connect this database with EC2 instance in that VPC in same AZ. Using free-tier for this.

## RDS:
1. Click on RDS service by AWS.
2. Create New database using RDS.
3. Select PostgreSQL

## Config:
1. Name of database
2. Master name for database-user
3. password.
4. select type of intance-configuations for database
5. storage
6. VPC
7. Subnet
8. etc.

## Connect:
1. Select EC2 Instance
2. VPC and SG
3. On EC2 install
