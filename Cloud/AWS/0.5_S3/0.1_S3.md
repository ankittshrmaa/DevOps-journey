# Simple Storage Service (Public Cloud)

Most popular and simple service of AWS.
S3 is popular because of 11 nine's, 99.999999999

### S3 benefits:
1. Scalable: No limit of storing data into S3, But one object size should be not more then 5TB.
2. Highly Available: AWS S3 available 24*7, with high speed Upload and Download.
3. Secure: The data we store in S3 is in encrypted format, polices, access control, permissions.
4. Cost effective: very cost effective then most of the cloud platforms, Also have HDD, SSD. choose as we need called classes.
5. Reliable: 11 9's means its reliable 99.999999999% so no risk of any reliability.
 S3 able to store images(.png,.jpeg,.pdf), Files, folders, MP3, Mp4, Csv-datasets and many more...

 ### S3 in DevOps: 
 In DevOps S3 is mostly use to store Logs, Backups, data-logs, configuration files.
